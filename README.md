# Personal Expense Tracker

A simple Expense Tracker project built using Python.  
This project helps users manage daily expenses by adding, editing, deleting, and viewing expenses along with budget alerts.

---

##  Overview

This program allows users to set their personal details and monthly budget.  
They can then record various expenses and keep track of their spending.  
The program warns the user when their total spending exceeds the budget.

The goal of this project is to understand basic Python concepts such as:
- Variables
- Conditional statements
- Loops
- Functions
- Lists & Dictionaries
- User input handling

---

##  Features

1. User details input (Name & Age)  
2. Set monthly budget  
3. Add new expenses  
3. View all recorded expenses  
4. Edit existing expense  
5. Delete an expense  
6. Search expense by name  
7. Calculate and display total spending  
8. Budget warning when overspending  

---

##  Technologies Used

- **Python**
- Works on any OS where Python is installed (Windows, macOS, Linux)

##  How to Install & Run

1. Install Python on your system (if not already installed)
   https://www.python.org/downloads/

3. Download or clone this repository

   git clone https://github.com/your-username/VITyarthi-Project.git

4. Go to the project folder

   cd VITyarthi-Project

5. Run the program

   python expense_tracker.py
