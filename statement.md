##  Problem Statement

Many individuals struggle to track their daily expenses and often exceed their budget due to a lack of proper monitoring.  
This leads to financial mismanagement and difficulty in controlling expenditure.  
A simple and user-friendly tool is required to record expenses, review spending patterns, and stay within a predefined budget.

The Personal Expense Tracker solves this problem by helping users:
- Record and manage day-to-day expenses
- Monitor their total spending
- Stick to their monthly budget with alerts when overspending

---

##  Scope of the Project

This project focuses on creating a basic command-line financial tracking system that includes:
- User profile details
- Budget setup for a month
- Expense entry and management
- Ability to calculate and review total expenses

---

##  Target Users

This system is best suited for:
- Students wanting to manage their pocket money
- Individuals new to budgeting and expense tracking
- Beginners learning Python and basic project development
- Anyone who wants a lightweight tool to manually track expenses

---

##  High-Level Features

1. User Details (Name & Age)  
2. Set Monthly Budget  
3. Add Daily Expenses  
4. View Complete Expense List  
5. Edit or Delete Recorded Expenses  
6. Search Expense by Name  
7. Automatic Total Expense Calculation  
8. Warning Message when Budget is Exceeded  

All features are implemented without external libraries, making the program very lightweight and easy to run on any system with Python.
